🚀 QUICK SETUP (3 STEPS):

1. RUN THE MIGRATION
   ┌─────────────────────────────────────────────────────────────────┐
   │ cd db/migrations                                                 │
   │ php run_designations_migration.php                               │
   └─────────────────────────────────────────────────────────────────┘

   - 20251203_create_designations_table.sql
   - 20251203_add_designation_to_faculty_profiles.sql

2. VERIFY IN ADMIN PANEL
   ┌─────────────────────────────────────────────────────────────────┐
   │ Go to: Admin > Settings > Manage Master Lists                   │
   │ Click: "Designations" tab                                       │
   │ You should see: Dean, Program Chair, Department Head, etc.      │
   └─────────────────────────────────────────────────────────────────┘

3. START USING
   ┌─────────────────────────────────────────────────────────────────┐
   │ Go to: Admin > Create Faculty                                   │
   │ Look for: "Designation" dropdown (under Employment Info)        │
   │ Select: Any designation from the list                           │
   └─────────────────────────────────────────────────────────────────┘

Settings > Manage Master Lists > Designations > Type name > Add

EDIT A DESIGNATION:
Settings > Manage Master Lists > Designations > Click "Edit" > Change name > Save

DELETE A DESIGNATION:
Settings > Manage Master Lists > Designations > Click "Delete" > Confirm

ASSIGN TO FACULTY (Batch):
Create Faculty > Batch Upload > Download Template > Fill "Designation" column > Upload
